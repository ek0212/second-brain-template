# Attachment Rules

- Keep attachments with their source bundle when possible.
- Use `attachments/` for shared files that are not tied to one raw source.
- When a wiki page depends on an attachment, link to it from the page or source note.
- Avoid renaming attachments unless doing so improves traceability.

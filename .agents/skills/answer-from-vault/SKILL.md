# Answer From Vault

Use this skill when answering a user question using the second brain.

## Workflow

1. Search `wiki/` first.
2. Use `raw/processed/` and `raw/unprocessed/` as backup evidence when needed.
3. Prefer synthesis over raw excerpts.
4. Mention uncertainty when notes are incomplete or conflicting.
5. Link to relevant curated pages.
6. If the answer creates a durable synthesis, offer to file or update a wiki page.

## Response Style

- Lead with the answer.
- Cite the strongest local evidence.
- Keep raw quotations short.
- Distinguish what the vault says from your inference.

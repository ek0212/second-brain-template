# Synthesize Topic

Use this skill when building or refreshing a topic page from multiple notes.

## Workflow

1. Search curated pages first.
2. Search raw notes as backup evidence.
3. Identify the best existing page to update, or choose a new title if needed.
4. Merge overlapping material into one concise synthesis.
5. Preserve nuance, uncertainty, decisions, and contradictions.
6. Link related pages instead of duplicating full content.
7. Add unresolved items to `Open Questions`.
8. Update `wiki/index.md` if the topic becomes a useful hub.

## Page Shape

Prefer:

```markdown
---
aliases: []
tags: []
status: draft
sources: []
---

# Page Title

## Summary

## Key Points

## Details

## Open Questions

## Related

## Sources
```

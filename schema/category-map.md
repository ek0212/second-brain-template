# Category Map

Customize this file for each vault. The default categories below are placeholders,
not a universal theory of knowledge.

## Default Curated Folders

```text
wiki/
  areas/
  projects/
  people/
  places/
  references/
  decisions/
  archive/
```

## How To Choose A Folder

Choose the folder based on why the page would be reopened.

- Reopen for an ongoing responsibility: `areas/`
- Reopen for a bounded effort with an outcome: `projects/`
- Reopen for notes about a person or relationship: `people/`
- Reopen for notes about a location: `places/`
- Reopen for reusable knowledge: `references/`
- Reopen for a choice, tradeoff, or conclusion: `decisions/`
- Reopen only as historical context: `archive/`

If a note fits several folders, choose one primary page and add links from related
pages. Prefer links over duplication.

## Adapting Categories

Replace the defaults with folders that match the vault's purpose. Examples:

- A work vault might use `projects/`, `meetings/`, `people/`, `decisions/`, `processes/`, and `research/`.
- A writing vault might use `characters/`, `worlds/`, `scenes/`, `drafts/`, `themes/`, and `research/`.
- A health vault might use `medical/`, `labs/`, `symptoms/`, `providers/`, `nutrition/`, and `training/`.

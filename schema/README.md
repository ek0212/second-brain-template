# Schema

This folder is the human-readable source of truth for the vault.

- `wiki-schema.md` explains the operating model.
- `category-map.md` defines the current curated folders.
- `page-template.md` is the default curated page shape.
- `ingest-workflow.md` describes how raw notes become wiki pages.

When the organization stops fitting real notes, update the schema before moving
large parts of the wiki.
